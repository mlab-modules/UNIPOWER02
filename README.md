# UNIPOWER02A
<!--- Emodule --->

<!--- subtitle --->
Universal power supply module from barrel DC connector and VAGO terminal block to MLAB modules. The module contains a variable fuse, switch and LED indicator. Module can be used either for 5V and 12V.
